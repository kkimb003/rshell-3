#Rshell - a bash terminal emulator
Rshell emulates all behaviors a normal bash shell would do

##Installation
` ``$ git clone https://github.com/sthak004/rshell.git`` `
` ``$ cd rshell`` `
` ``$ git checkout hw0`` `
` ``$ make`` `
` ``$ bin/rshell`` `

Bug and Limitations
*only single command currently work. As in `ls` or `ls-a`.
